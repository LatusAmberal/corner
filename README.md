📘 一隅·aCorner
一个纯净、可定制的 AI 角色扮演聊天应用，主要是自用为主的梦角养成器。
目前v1版本只包含聊天内容，后续会加上包括但不限于专注/模拟朋友圈/模拟视频通话等等功能，等我有时间了就改。

目前的代码内容全部都是ai修改提供的，本人完全代码小白，做这个自用为主


🚀 快速开始
1️⃣ 下载文件
直接下载 index1.0.html 文件，双击即可在浏览器中运行（无需安装任何依赖）。

2️⃣ 配置 API
点击左侧齿轮图标进入「设置」。

选择 API 服务商（推荐 DeepSeek，注册即送免费额度）。

填写 API 地址和你的 API Key。

点击「测试连接」，成功后会显示 ✅。

保存配置。

3️⃣ 设定角色
点击左侧「人物」图标。

填写角色的世界观、姓名、性格、背景故事、对话风格等。

点击「保存人物设定」。

4️⃣ 开始聊天
在输入框中发送消息，AI 就会按照你设定的人设进行回复。

🔧 自定义 API 服务商
服务商	默认 API 地址	说明
DeepSeek	https://api.deepseek.com/v1/chat/completions	国内可用，注册送额度
OpenAI	https://api.openai.com/v1/chat/completions	需海外网络
OpenRouter	https://openrouter.ai/api/v1/chat/completions	聚合多家模型，有免费模型
Groq	https://api.groq.com/openai/v1/chat/completions	免费但限制频率
自定义	任意兼容 OpenAI 格式的地址	

📦 数据导出与迁移
点击左侧「数据库」图标进入数据管理页面：

查看各存储项占用空间。

点击「导出全部数据」下载 JSON 备份文件。

在新设备上点击「导入数据文件」即可恢复所有设置与角色数据。
